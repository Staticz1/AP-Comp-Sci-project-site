Drawing App 1.0

Drawing app 1.0

The first project is a drawing app My partner and I made in AP computer science class. The version shown is the V1 version of the app. In this version, you get a decent variety of options to choose from. You can draw (with multiple colors), erase, and create shapes using the buttons assigned to them! What made this app hard to make was trying to get all the buttons to work properly. Some of them wouldn’t work how my partner and I wanted them to, and sometimes, they stopped working entirely! This app still has a long way to go, but I think it’s a decent start.

Scratch Game

Irridated (game)

The second project I’d like to show you is my own game that my partner and I made duing programming and robotics class. The game is called “Irradiated”, and it’s a third-person apaocalyptic game where you, the player, who is a survivor of a thermonuclear war, must try to find shelter while also fighting off mutated monsters. This was the first big scratch game I made, and this took a lot of work and time to make, especially since I was basically learning everything as I worked! It was a struggle, but the results turned out great. Fun fact: The game has a total of 2,046 blocks of code!

Soldering

Here is a hand-made project I made. This little device is called the knight rider, and this was made by soldering! This device shows a repeating pattern of the light circulating through the lightbulbs on the board. I chose soldering for my genius hour, and it was pretty interesting to make this. Although it’s something small and simple, it took a lot of careful steps in order to make it correctly. If a little piece were to be put incorrectly, or if it went missing, the entire project had to be redone. This took a lot of time to do, as we were very slow during the process of making it, making sure we didn’t lose any pieces. I learned a lot about soldering during the project, as well as how creative the kits got. This was truly a fun one to do!
